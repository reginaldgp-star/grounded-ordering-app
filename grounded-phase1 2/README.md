# Grounded Phase 1 Ordering App

This is the starter app for the Grounded live coffee ordering system.

## What it does
- Mobile ordering screen
- Barista live queue
- Mark orders as Start / Ready / Complete
- Live stats and end-of-event report
- Uses Firebase Firestore for multi-device syncing

## Setup
1. Install Node.js
2. Run `npm install`
3. Create a Firebase project
4. Enable Firestore Database
5. Copy your Firebase web app config into `src/firebase.js`
6. Run `npm run dev`
7. Deploy to Firebase Hosting, Netlify, or Vercel

## Menu
- The Grounded – Hot Vanilla Latte
- The Watchman – Iced Vanilla Latte
- The Daily Bread – Americano / Black Coffee
- The Good News – Mocha Latte
